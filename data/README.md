# 中国象棋棋谱数据库 (Xiangqi Game Database)

## 📊 数据概览
- **项目范围**: Phase 3 - 棋谱数据收集
- **总局数**: 已收集 321 局
- **版本**: v1.0 (2026-02-08)

## 📁 目录结构
- **/ancient**: 古谱 json (烂柯神机, 橘中秘, 梅花谱, 适情雅趣)
- **/modern**: 现代大师赛 json (胡荣华, 王天一, 许银川)
- **/docs**: PGN 格式报告

## 🧬 数据格式 (Schema)
所有数据遵循标准 JSON 格式，核心字段如下：
- `id`: 棋局唯一标识
- `moves`: 着法数组
- `opening`: 开局名称
- `era`: 历史年代
